# CC-11: AVI / ETC - Sistema de Peajes Electrónicos

## 8.11.1 Resumen Ejecutivo

El Sistema de Peajes Electrónicos (AVI - Automatic Vehicle Identification / ETC - Electronic Toll Collection) constituye una solución tecnológica avanzada para la **Automatización de Transacciones** en infraestructuras de transporte de peaje. Este sistema permite procesar cobros de tasa de uso sin detener el movimiento vehicular, eliminando congestión en plazas de peaje y mejorando significativamente la experiencia del usuario.

La capacidad de **Automatización de Transacciones** mediante el procesamiento de cobros sin detener el flujo vehicular es fundamental para maximizar la capacidad de la vía, reducir tiempos de viaje, disminuir emisiones contaminantes y optimizar los costos operativos de la concesión. El sistema AVI/ETC integra tecnologías de identificación automática (RFID, DSRC), procesamiento de pagos electrónicos, detección de vehículos y sistemas de videocontrol para proporcionar una solución integral de cobro de peaje.

## 8.11.3 Descripción Funcional y Técnica

### 3.1 Propósito del Sistema

El AVI/ETC tiene como propósito fundamental automatizar el cobro de peajes para:

- **Eliminación de congestión**: Paso continuo sin detención en peajes

- **Optimización de costos**: Reducción de personal y mantenimiento de cabinas

## 8.11.11 Casos de Uso

### CASO DE USO 1: Paso Fluido en Hora Pico

**Actor Principal:** Usuario del sistema vial con OBU instalado

**Flujo Principal:**
1. Usuario se aproxima a plaza de peaje en hora pico (volumen alto)
2. Sistema detecta vehículo y activa antena DSRC
3. OBU del vehículo responde con identificación única
4. Sistema valida cuenta activa con saldo suficiente
5. Transacción se procesa en < 500 ms
6. Usuario pasa sin detenerse a velocidad normal (60 km/h)
7. LED verde confirma transacción exitosa

**Valor Generado:** Ahorro de 10-15 minutos por peaje, reducción de estrés

---

### CASO DE USO 2: Detección de Evasión

**Actor Principal:** Operador de control de peaje

**Flujo Principal:**
1. Vehículo ingresa a carril ETC sin OBU válido
2. Antena DSRC no detecta respuesta del vehículo
3. Sistema activa cámara ANPR para capturar placa
4. Reconocimiento de placa genera alerta de evasión
5. Sistema consulta lista negra de evasores reincidentes
6. Se genera registro de infracción con imagen y timestamp
7. Vehículo se añade a lista de seguimiento para recuperación

**Valor Generado:** Control de evasión, recuperación de ingresos

---

### CASO DE USO 3: Recarga Automática de Saldo

**Actor Principal:** Usuario con cuenta ETC registrada

**Flujo Principal:**
1. Usuario configura recarga automática con tarjeta de crédito
2. Sistema monitore saldo de cuenta periódicamente
3. Saldo cae por debajo de umbral configurado ($10,000)
4. Sistema inicia recarga automática por $50,000
5. Pasarela de pagos procesa transacción con banco
6. Saldo se actualiza en tiempo real
7. Usuario recibe notificación de recarga exitosa

**Valor Generado:** Conveniencia, nunca sin saldo, flujo continuo

---

### CASO DE USO 4: Tarifa Dinámica por Congestión

**Actor Principal:** Gestor de tráfico de la concesión

**Flujo Principal:**
1. Sistema TMS detecta congestión en tramo principal
2. Se activa política de tarifa dinámica para desviar tráfico
3. Sistema ETC aplica descuento del 30% en hora valle
4. Se incrementa tarifa 20% en hora pico de congestión
5. Usuarios reciben información de tarifas en app móvil
6. Se observa migración de 15% de tráfico a horarios alternos
7. Se reduce congestión promedio en 25%

**Valor Generado:** Gestión de demanda, optimización de red vial

---

### CASO DE USO 5: Integración con App Móvil

**Actor Principal:** Usuario con smartphone

**Flujo Principal:**
1. Usuario descarga app de la concesión vial
2. Registra vehículo y vincula OBU virtual (BLE)
3. App muestra saldo, historial de viajes y recargas
4. Usuario recibe alertas de promociones y descuentos
5. En peaje, app usa BLE para comunicación con antena
6. Transacción se procesa igual que con OBU físico
7. Usuario recibe notificación push con detalle del peaje

**Valor Generado:** Experiencia digital integrada, conveniencia adicional
