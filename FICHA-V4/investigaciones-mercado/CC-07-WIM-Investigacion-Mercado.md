# Investigación de Mercado - CC-07: WIM (Weigh-in-Motion)

## 1. Especificaciones Técnicas

### 1.1 Descripción del Equipo
| Campo | Especificación |
|-------|----------------|
| **Código** | CC-07 |
| **Nombre Completo** | Weigh-in-Motion / Pesaje en Movimiento |
| **Categoría** | Sistema de Pesaje Vehicular Dinámico |
| **Aplicación Principal** | Pesaje de vehículos en movimiento para control de peso, clasificación y protección de infraestructura vial |

### 1.2 Especificaciones Funcionales
| Parámetro | Especificación |
|-----------|----------------|
| Rango de peso por eje | 0-30,000 kg |
| Precisión de pesaje | Clase 0.5A (Costa 99.5%), Clase B+ (Costa 99%), Clase C (Costa 96%) |
| Velocidad de operación | 5-120 km/h |
| Clasificación vehicular | Por esquema de ejes (FHWA 13 clases o personalizado) |
| Distancia entre ejes | Detección automática |
| Conteo vehicular | 100% de vehículos |
| Almacenamiento de datos | Local y transmisión a centro de control |
| Comunicaciones | Ethernet, RS-485, RS-232, 4-20mA |
| Alimentación | 110-240 VAC / 24 VDC |

### 1.3 Especificaciones Técnicas Detalladas
| Característica | Valor |
|----------------|-------|
| Tecnología de sensor | Piezoeléctrico (Línea), Puntual (Capacitivo), Bending Plate, Array |
| Tiempo de respuesta | <1 ms |
| Temperatura operación | -40°C a +70°C |
| Humedad | 0-100% RH |
| Protección IP | IP68 (sensores enterrados) |
| Ciclo de vida del sensor | >100 millones de ejes |
| Certificaciones | OIML R134, ASTM E1318, COST 323 |

### 5.2 Recomendación
Basado en las especificaciones del proyecto y estándares internacionales:

1. **Opción Primaria**: **Kistler Lineasens C** - Líder mundial, precisión Clase 0.5A, tecnología probada, certificaciones OIML/ASTM
2. **Opción Alternativa**: IRD - Sistema integral completo, buena relación costo-beneficio, amplia experiencia
3. **Opción Económica**: Soluciones basadas en sensores piezoeléctricos genéricos - Para clasificación vehicular sin alta precisión de peso

### 5.3 Justificación Kistler
- Líder absoluto en el mercado de WIM con sensores piezoeléctricos de cuarzo
- Precisión Clase 0.5A (Costa 323) - la más alta del mercado
- Vida útil superior a 100 millones de ejes (sin mantenimiento)
- Certificado OIML R134, ASTM E1318, COST 323
- Tecnología probada en proyectos de peaje y control de peso mundialmente
- Sensores compatibles con instalación superficial o empotrada
