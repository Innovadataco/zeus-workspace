## RESUMEN EJECUTIVO

### ¿Qué es PM²?

PM² (Project Management Methodology) es una **metodología de gestión de proyectos desarrollada por la Comisión Europea** para sus instituciones, administraciones públicas y contratistas. Es una metodología:

- **Abierta**: Disponible bajo licencia Creative Commons (Open PM²)
- **Adaptable**: Puede ajustarse a cualquier tipo de organización
- **Integrada**: Combina elementos de PMI/PMBOK, PRINCE2, y prácticas ágiles
- **Probada**: Utilizada en miles de proyectos de la UE desde 2008

### ¿Por qué PM² para Innovadataco?

| Ventaja | Beneficio para Innovadataco |
|---------|---------------------------|
| **Estándar europeo** | Credibilidad ante clientes institucionales |
| **Open Source** | Sin costos de licenciamiento |
| **Adaptable** | Se ajusta a consultorías y proyectos tecnológicos |
| **Documentada** | Plantillas listas para usar |
| **Escalable** | Desde proyectos pequeños hasta grandes consultorías |

### Ciclo de Vida en una Línea

**Inicio → Planificación → Ejecución → Cierre**
*(con Seguimiento y Control transversal a todo el ciclo)*

### Fase Ágil para Productos Tecnológicos

PM² Agile permite combinar la **estructura institucional de PM²** con la **flexibilidad ágil** para desarrollo de software y productos tecnológicos.
