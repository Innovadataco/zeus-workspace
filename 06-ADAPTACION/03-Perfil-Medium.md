# 6.3 — Perfil Medium para Innovadataco

## Definición

Perfil **Medium** para proyectos de duración y complejidad media. Balance entre estructura y agilidad.

---

## Características

| Aspecto | Especificación |
|---------|--------------|
| **Duración** | 2-6 meses |
| **Equipo** | 3-6 personas |
| **Presupuesto** | $10K - $100K USD |
| **Complejidad técnica** | Media |
| **Riesgos** | Medios, gestionables |
| **Stakeholders** | Múltiples, identificados |

---

## Ciclo de Vida Completo (con adaptaciones)

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  INICIO │───→│   PLAN  │───→│  EJECUCIÓN│───→│ CIERRE  │
│ (1-2 sem)│    │ (2-4 sem)│    │ (2-4 mes)│    │ (1-2 sem)│
└─────────┘    └─────────┘    └─────────┘    └─────────┘
       LpP           LpE           LpC
```

---

## Artefactos Estándar

### Fase 1: INICIO
- OPM2-01: Solicitud de Inicio
- OPM2-02: Caso de Negocio (light)
- OPM2-03: Acta de Constitución

### Fase 2: PLANIFICACIÓN
- OPM2-04: Manual del Proyecto (light)
- OPM2-05: Matriz de Partes Interesadas
- OPM2-06: Plan de Trabajo
- OPM2-11: Plan de Gestión de Requisitos
- OPM2-13: Plan de Gestión de Riesgos (light)
- OPM2-16: Plan de Gestión de la Comunicación

### Fase 3: EJECUCIÓN
- OPM2-17: Agenda de Reunión
- OPM2-18: Acta de Reunión
- OPM2-19: Informe de Situación (quincenal)
- OPM2-22: Informe de Progreso (semanal)
- OPM2-23: Registro de Riesgos
- OPM2-24: Registro de Incidencias

### Fase 4: CIERRE
- OPM2-33: Informe de Fin de Proyecto

## Reuniones Estándar

| Reunión | Frecuencia | Duración | Participantes |
|---------|-----------|----------|---------------|
| Kick-off Inicio | Una vez | 1 hora | CDP completo |
| Reunión LpP | Una vez | 1 hora | CDP |
| Planning | Una vez | 2-4 horas | DP + EDP |
| Reunión LpE | Una vez | 1 hora | CDP |
| Stand-up / Check-in | Semanal | 30 min | DP + EDP |
| Revisión progreso | Quincenal | 1 hora | CDP |
| Reunión LpC | Una vez | 1 hora | CDP |
| Cierre | Una vez | 1-2 horas | CDP + EDP |

---

## Plantillas Simplificadas

Para proyectos Medium en Innovadataco, las plantillas pueden ser:
- **Caso de negocio:** 3-5 páginas (no 20+)
- **Manual del proyecto:** 5-10 páginas
- **Plan de trabajo:** Spreadsheet con Gantt simple
- **Informes:** 1-2 páginas máximo
