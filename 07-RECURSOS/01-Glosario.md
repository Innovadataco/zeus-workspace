# 7.1 — Glosario de Términos PM²

## Términos Fundamentales

| Término | Definición |
|---------|------------|
| **PM²** | Project Management Methodology — Metodología de Gestión de Proyectos de la Comisión Europea |
| **Fase** | Periodo dentro del ciclo de vida del proyecto (Inicio, Planificación, Ejecución, Cierre) |
| **Phase Gate** | Punto de revisión formal al final de cada fase (LpP, LpE, LpC) |
| **Artefacto** | Documento, plantilla o registro utilizado en la gestión del proyecto |
| **Entregable** | Producto, servicio o resultado tangible del proyecto |
| **Alcance** | Conjunto de trabajo que debe realizarse para entregar el producto |
| **Stakeholder** | Parte interesada — persona o entidad afectada por el proyecto |
| **Riesgo** | Evento incierto que, si ocurre, tiene efecto positivo o negativo |
| **Incidencia** | Problema o situación que requiere atención y resolución |
| **Cambio** | Modificación aprobada al alcance, cronograma o presupuesto |
| **Caso de negocio** | Justificación de la inversión en el proyecto |
| **Acta de constitución** | Documento que autoriza formalmente el proyecto |
| **WBS** | Work Breakdown Structure — Estructura de Desglose de Trabajo |
| **RASCI** | Matriz de responsabilidades (Responsible, Accountable, Support, Consult, Inform) |
| **CDP** | Comité de Dirección del Proyecto |
| **PP** | Propietario del Proyecto |
| **DP** | Director de Proyecto |
| **RN** | Responsable de Negocio |
| **EDP** | Equipo de Desarrollo del Proyecto |
| **PS** | Proveedor de Soluciones |
| **GIN** | Grupo de Implementación en el Negocio |

## Términos Ágiles

| Término | Definición |
|---------|------------|
| **Sprint** | Iteración de tiempo fijo (1-4 semanas) con entregable funcional |
| **Backlog** | Lista priorizada de requisitos o trabajo pendiente |
| **User Story** | Descripción de una funcionalidad desde perspectiva del usuario |
| **Story Points** | Unidad de medida relativa del esfuerzo |
| **Velocity** | Cantidad de trabajo completado por sprint |
| **Burndown** | Gráfico de trabajo pendiente vs. tiempo |
| **Definition of Done** | Criterios para considerar un item terminado |
| **Product Owner** | Rol ágil equivalente al PP — define visión y prioriza |
| **Scrum Master** | Facilitador del proceso ágil |
| **Retrospective** | Reunión de reflexión y mejora continua |
| **Stand-up** | Reunión diaria corta de sincronización |
| **Kanban** | Tablero visual de flujo de trabajo |

| **QA** | Quality Assurance |
| **ROI** | Return on Investment |
