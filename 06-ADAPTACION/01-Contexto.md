# 6.1 — Contexto de Innovadataco

## Perfil de la Organización

| Característica | Descripción |
|----------------|-------------|
| **Nombre** | INNOVADATACO |
| **Sector** | Tecnología, consultoría, desarrollo de productos |
| **Modelo** | Human + AI Orchestration |
| **Tamaño** | Equipo reducido, alta especialización |
| **Clientes** | Sector público, instituciones, empresas |
| **Fortalezas** | Agilidad, innovación, capacidad técnica |
| **Desafíos** | Recursos limitados, múltiples roles por persona |

---

## Tipos de Proyectos en Innovadataco

| Tipo | Características | Metodología Recomendada |
|------|-----------------|------------------------|
| **Consultoría estratégica** | Requisitos definidos, entregables documentales | PM² Tradicional (Light/Medium) |
| **Desarrollo de productos** | Requisitos emergentes, tecnología nueva | PM² Agile (Medium) |
| **Proyectos institucionales** | Gobernanza estricta, regulatorios | PM² Tradicional (Medium/Heavy) |
| **Consultoría + Desarrollo** | Fases iniciales definidas + ejecución ágil | PM² Híbrido |
| **Experimentos/POCs** | Alto riesgo, aprendizaje rápido | PM² Agile (Light) |

---

## Adaptación de Roles para Innovadataco

### Roles Compartidos (Realidad de Equipos Pequeños)

| Rol PM² | Persona en Innovadataco | Responsabilidades |
|---------|------------------------|-------------------|
| **PP** | Jelkin (CEO) | Visión, caso de negocio, aprobaciones |
| **DP** | Jelkin / Asignado por proyecto | Gestión diaria, coordinación |
| **RN** | Jelkin / Diana (según proyecto) | Requisitos, aceptación, transición |
| **EDP** | Equipo técnico + ZEUS (IA) | Desarrollo, entregables |
| **PS** | Juan (Innovación) / Externo | Arquitectura técnica |

### ZEUS como Miembro del EDP

ZEUS puede asumir funciones de:
- **Documentación:** Generar artefactos PM² (informes, actas, planes)
- **Análisis:** Análisis de riesgos, stakeholders, requisitos
- **Seguimiento:** Monitorear progreso, generar burndowns
- **Comunicación:** Preparar presentaciones, resúmenes ejecutivos

## Perfiles de Proyecto para Innovadataco

| Perfil | Duración | Complejidad | Documentación | Ejemplo |
|--------|----------|-------------|-------------|---------|
| **Light** | < 2 meses | Baja | Mínima | Diagnóstico rápido, POC |
| **Medium** | 2-6 meses | Media | Estándar | Desarrollo de MVP, consultoría completa |
| **Heavy** | > 6 meses | Alta | Extensiva | Proyecto institucional, producto enterprise |
