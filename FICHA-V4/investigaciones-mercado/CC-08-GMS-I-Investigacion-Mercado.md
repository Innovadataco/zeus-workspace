## 1. Especificaciones Técnicas

### 1.1 Descripción del Equipo
| Campo | Especificación |
|-------|----------------|
| **Código** | CC-08 |
| **Nombre Completo** | Geotechnical Monitoring System - Instrumentation |
| **Categoría** | Sistema de Monitoreo Geotécnico e Instrumentación |
| **Aplicación Principal** | Monitoreo de estabilidad de taludes, deslizamientos, presas, túneles y estructuras geotécnicas |

### 1.2 Especificaciones Funcionales - Inclinómetros
| Parámetro | Especificación |
|-----------|----------------|
| Tipo de sensor | MEMS (Micro-Electro-Mechanical Systems) |
| Rango de medición | ±15° (vertical), ±30° (horizontal) |
| Resolución | 0.005 mm por 500 mm (2 arc sec) |
| Precisión | ±0.004° (±13.5 arc sec, ±0.07 mm/m) |
| Repetibilidad | ±0.002° (±7.2 arc sec) |
| Temperatura operación | -40°C a +85°C |
| Profundidad máxima | Hasta 200 m (según cable) |
| Diámetro de sonda | 25.4 mm (1 pulgada) |
| Distancia entre ruedas (wheelbase) | 0.5 m estándar |
| Comunicación | RS-485 Digital Bus (MODBUS RTU) |
| Alimentación | 5-15 VDC |
| Protección IP | IP68 (2 MPa) |

### 1.3 Especificaciones - Inclinómetros In-Place (IPI)
| Parámetro | Especificación |
|-----------|----------------|
| Tecnología | MEMS (acelerómetros) |
| Rango | ±15° desde vertical, ±15° desde 45°, ±15° desde horizontal |
| Resolución | 0.0005° (2 arc sec, 0.01 mm/m) |
| Precisión | ±0.004° (±0.07 mm/m) |
| Salida | RS-485 Digital Bus |
| Temperatura operación | -40°C a +85°C |
| Diámetro sensor | 28.6 mm (1.125 in) |
| Longitud gauge | 0.5 a 3 m configurable |

### 1.4 Especificaciones - Piezómetros
| Parámetro | Especificación |
|-----------|----------------|
| Tipo | Cuerda vibrante o piezoeléctrico |
| Rango de presión | 0-10 bar, 0-20 bar, 0-50 bar (configurable) |
| Resolución | 0.025% del rango |
| Precisión | ±0.2% FS |
| Salida | Frecuencia (cuerda vibrante) o 4-20mA |
| Temperatura | -20°C a +80°C |
