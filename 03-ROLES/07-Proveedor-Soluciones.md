# 3.7 — Proveedor de Soluciones (PS)

## Definición

El **Proveedor de Soluciones (PS)** es la persona o entidad que **provee recursos técnicos y soluciones** para el proyecto. Representa el lado "proveedor" o "técnico".

---

## Responsabilidades Clave

| Área | Responsabilidad |
|------|----------------|
| **Recursos** | Proveer recursos técnicos al proyecto |
| **Soluciones** | Asegurar viabilidad técnica de las soluciones |
| **Calidad técnica** | Garantizar estándares técnicos |
| **Externalización** | Gestionar contratistas si aplica |
| **Arquitectura** | Asegurar coherencia con arquitectura empresarial |

---

## Relación con PP

El PP define **qué** se necesita, el PS define **cómo** técnicamente se puede lograr.
