# 3.3 — Director de Proyecto (DP)

## Definición

El **Director de Proyecto (DP)** es la persona responsable de la **gestión diaria del proyecto**. Coordina al equipo, supervisa el progreso, gestiona riesgos y mantiene comunicación con todas las partes interesadas.

---

## Responsabilidades Clave

| Área | Responsabilidad |
|------|----------------|
| **Planificación** | Desarrollar y mantener el plan de trabajo |
| **Ejecución** | Coordinar y dirigir el trabajo del equipo |
| **Seguimiento** | Monitorear progreso, costos, calidad |
| **Riesgos** | Identificar, evaluar y gestionar riesgos |
| **Comunicación** | Mantener informadas a todas las partes interesadas |
| **Cambios** | Gestionar solicitudes de cambio |
| **Informes** | Producir informes de progreso y situación |
| **Cierre** | Coordinar cierre administrativo |

## Habilidades Requeridas

- Liderazgo y gestión de equipos
- Comunicación efectiva
- Gestión de conflictos
- Planificación y organización
- Gestión de riesgos
- Conocimiento de la metodología PM²
