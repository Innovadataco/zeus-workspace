# 4.4 — Listas de Control (CHL)

## OPM2-27 — Lista de Control de Salida de Fase

### Propósito
Verificar que todos los criterios de salida de una fase se han cumplido antes de proceder a la siguiente.

### Uso
Aplicar al final de cada fase (Inicio, Planificación, Ejecución) antes del respectivo Phase Gate (LpP, LpE, LpC).

## OPM2-30 — Lista de Control de Transición

### Propósito
Verificar que todos los elementos necesarios están listos para la transición del proyecto a operaciones.

### Secciones
1. **Documentación de operación**: Manuales, procedimientos, diagramas
2. **Formación**: Usuarios formados, materiales entregados
3. **Infraestructura**: Sistemas instalados, configurados, probados
4. **Soporte**: Acuerdos de soporte establecidos
5. **Datos**: Migración de datos completada y verificada
6. **Seguridad**: Accesos configurados, permisos definidos
7. **Monitoreo**: Métricas de operación definidas

---

## OPM2-31 — Lista de Control de Implementación en el Negocio

### Propósito
Verificar que la solución está lista para ser desplegada en el entorno de producción del negocio.

### Secciones
1. **Pruebas**: Todas las pruebas pasaron (unitarias, integración, sistema, aceptación)
2. **Rendimiento**: Métricas de rendimiento cumplen requisitos
3. **Seguridad**: Auditoría de seguridad completada
4. **Backup/Recuperación**: Procedimientos probados
5. **Comunicación**: Usuarios informados del despliegue
6. **Plan de rollback**: Procedimiento de reversión definido
7. **Soporte**: Equipo de soporte listo
